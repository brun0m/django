# Django Simple CRUD Project

Este é um projeto simples desenvolvido com Django para fins de treinamento, com um front-end básico em HTML, CSS e um pouco de Bootstrap. O objetivo deste projeto é demonstrar a implementação de um CRUD (Create, Read, Update, Delete) em um banco de dados SQLite.

## Funcionalidades

- **Login:** Tela de autenticação para usuários cadastrados.
- **Cadastro:** Tela de registro para novos usuários.
- **Home:** Página principal do usuário, onde pode ser realizada as ações:
  - Ver todos os usuários cadastrados.
  - Alterar os próprios dados.
  - Excluir a própria conta.
  - Sair.
- **Gerenciamento de Conta:**
  - Visualizar e alterar seus próprios dados.
  - Excluir a própria conta.

## Tecnologias Utilizadas

- **Back-end:** Django
- **Banco de Dados:** SQLite
- **Front-end:** HTML, CSS e Bootstrap
